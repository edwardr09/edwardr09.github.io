This web portfolio includes examples of my work in Ms. Chang's Web Design class.

Unit 2

n Unit 2, I learned how to create my webpage using HTML code. Learning HTML is essential towards web design, and in this unit I learned the basics of using HTML. Part of the process was being able to do a variety of things with HTML to improve my web pages.

Additionally, I had to participate in a group project where we had to create a video promoting CTE programs. Along the video, we created captioning as well as posted the video on our websites. This group project gave me a lot of experience on working with outside resources to help develop my website.

Some informative things I learned this unit:

  • Root folder
  • Captioning
  • How to link to websites
  • How to add videos to websites
  • Absolute and relative links

Unit 3

In this Unit, we went a little bit more in-depth with how CSS operates. To begin with, I worked with color and typography more in order to make my website more appealing to visitors. I learned the difference of classes and IDs and how to utilize them to make work easier. Additionally, I learned what the box model is and how useful it can be for web designers like myself.

Here are the major things I learned this unit:

  • Color and typography
  • Classes and IDs
  • Page Layout Techniques
  • The Box Model

Unit 4

Unit Four mostly revolved around artwork and designing your website. For example, I worked with creating a banner and a background image for my website. Additionally, I added a photo album to my website to share to the public, as well as buttons that I created myself for visitors to utilize.

Here are the major things I learned this unit:

  • How to create a button
  • How to add images to your website using HTML/CSS
  • How to utilize graphic designing programs for web design
  • The importance of dimensions and size in web designing

Unit 5

Unit Five was an introduction to Javascript and how it is essential for both websites and web designers. As a way to introduce Javascript, we created a clocked powered by Javascript. After that, we used our CSS and HTML skills in order to put the clock on our website and design it.

Here are the major things I learned this unit:

  • How Javascript impacts websites
  • Some basics of Javascript
  • How to create a clock using Javascript
  • How you can use CSS and HTML for Javascript

Unit 6

Unit Six introduced us to forms and surveys. The previous unit introduced us to Javascript, and for this unit we had to use what we learned and more to create a form for our website. Additionally, we had to validate our websites using resources on the Internet. This was a large help for me because it allowed me to see what errors I have on my website, despite how simple or complicated it is. After creating our forms, we had people from other classes test our website and see if we had any errors.

Here are the major things I learned this unit:

  • Creating forms using Javascript
  • Receiving criticism from others
  • Validating websites
  • Using Google Forms

Unit 7

Unit Seven was short but essential for web developers. This unit introduced us to web hosting and web services. We learned how web hosting works and why it's important for web develoeprs to know this type of information especially when having clients.

Here are the major things I learned this unit:

  • Different web services like GoDaddy, etc.
  • What Github is
  • How to publish your website on the web
